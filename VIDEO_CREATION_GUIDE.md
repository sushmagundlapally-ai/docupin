### Option 1: Screencasting with Voiceover (Recommended)
**Best for:** Professional demo videos, tutorials, product launches

**Tools Needed:**
- Screen recorder (QuickTime, OBS, Loom)

**Process:**

   - Read `DEMO_SCRIPT.md` 
   - Add 10-15 sample bookmarks to your app
   - Practice the demo flow
   - Write your voiceover script (or use provided script)

2. **Record Screen** (20 minutes)
   - Open DocuPin in browser
   - Record your screen following demo script scenes
   - Take multiple takes of tricky parts
   - Don't worry about mistakesâ€”you'll edit later

3. **Record Voiceover** (30 minutes)
   - Use `DEMO_SCRIPT.md` talking points
   - Record in a quiet room
   - Speak slowly and clearly
   - Record in sections (easier to edit)

4. **Edit** (1-2 hours)
   - Import screen recording and audio
   - Sync voiceover with visuals
   - Add text overlays for key points
   - Add transitions between scenes
   - Export as MP4 (1920x1080, 30fps)

**Total Time:** ~2.5-3 hours

## ðŸŽ¨ Visual Best Practices

### Screen Recording
- **Cursor highlighting:** Enable cursor spotlight in recording software
- **Smooth movements:** Move mouse slowly and deliberately
- **Click clearly:** Pause briefly before/after clicks
- **Zoom in:** For small text, zoom browser to 110-125%
- **Highlight areas:** Use annotations or circles to draw attention

### Text Overlays
- **Font:** Use DocuPin's brand font (Inter) or similar sans-serif
- **Colors:** Match DocuPin's palette (#4b6cb7, #182848)
- **Size:** Large enough to read on mobile (minimum 24pt)
- **Position:** Lower third (doesn't cover UI)
- **Duration:** 3-5 seconds per text

### Transitions
- **Simple:** Fade or dissolve (avoid flashy transitions)
- **Consistent:** Use same transition throughout
- **Quick:** 0.5-1 second duration

### Pacing
- **Slow down:** 30% slower than normal speaking
- **Pause:** 1-2 seconds after major points
- **Breathe:** Don't rushâ€”viewers need time to process

## âœ‚ï¸ Editing Tips

### Cuts and Timing
- **Remove "um"s and pauses:** Makes you sound more professional
- **Speed up mouse movements:** 1.2-1.5x for boring parts
- **Jump cuts:** Cut between major sections
- **B-roll:** Insert relevant visuals during voiceover

### Music
- **Volume:** 10-20% of voiceover (subtle background)
- **Style:** Upbeat, modern, corporate (not distracting)
- **Licensing:** Use royalty-free (YouTube Audio Library, Epidemic Sound)

### Captions
- **Always include:** 80% of social videos watched muted
- **Auto-generate:** YouTube/LinkedIn can create them
- **Edit for accuracy:** Fix any transcription errors
- **Style:** Yellow text with black outline (most readable)

### Color Correction
- **Brightness:** Ensure UI is clearly visible
- **Contrast:** Make text pop
- **Saturation:** Slight boost makes it more engaging
